# Payture API V1

This library provides a wrapper to Version 1 of the Payture API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'payture-api-v1'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install payture-api-v1
